## Архитектурные решения

### 1. Модульность
Каждая функциональная область вынесена в отдельное приложение:
- `core` — основные сущности
- `payments` — финансы
- `interactions` — коммуникации
- `lessons` — расписание
- `tasks` — планирование

### 2. Система прав
- Декораторы `@login_required`, `@role_required`, `@permission_required`
- Проверка прав в шаблонах через контекст
- Гибкая настройка через `RoleAssignment`

### 3. Glassmorphism-дизайн
- CSS-переменные для темизации
- Переиспользуемые компоненты
- Адаптивная верстка

### 4. Автоматизация
- Сигналы для обновления статусов
- Автоматическая генерация номеров счетов
- Триггеры для расчёта времени
